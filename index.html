About Kuznix

Kuznix is an independent Linux distribution built entirely from scratch using LFS, BLFS, and supplemental guides. We compile everything ourselves, with full multilib and gaming support.

Future Plan

  • Complete KDE, GNOME, LXQt desktop environments
  • WebKitGTK and Qt6-webengine integration
  • Own package system (.kzpkg) and ISO builds
  • Self-hosted GitLab, CI/CD, forums, and daily builds
  • Full gaming support and multimedia applications
